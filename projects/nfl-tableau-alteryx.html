NFL Tableau + Alteryx

Data-prep and visualization workflow analyzing deep pass rates and scoring efficiency.

Overview

This project explores relationships between offensive play-calling and efficiency outcomes using Alteryx for data preparation and Tableau for interactive visualization.

Approach

  • Cleaned and merged NFL play-by-play datasets in Alteryx.
  • Built visual dashboards highlighting deep-pass frequency vs. Expected Points Added (EPA).
  • Compared outcomes across game quarters and field positions.

Key Takeaways

  • Showed clear trend between explosive plays and scoring outcomes.
  • Demonstrated the value of blending play-level analytics with visualization for communication.
